这是一段模拟机器学习的项目，主要内容是加州房价的线性预测。（B站上其实很多教程，但是菜鸡一个没看懂。）这是一个经典问题，包括基础层（数据处理），单一特征选择（也就是时间线里的advanced层，但是后面又大改了一遍），最后是扩展层（包括极端值分析和地理位置的热力图）。整个项目基本还原了机器学习的流程，虽然结果怪怪的吧，但是我也没招了（）

第一部分的基础层，包含data_prep.py, 加载数据库（这个也怪怪的，可能是环境配置的问题，但是pycharm有点难用，而且不会接git，所以还是在vscode里写的），数据准备阶段写了很多函数，包括缺失值，MSE、MAE的计算（这个作业要求也很诡异，所以后面大改了），最后就是基本的8、2划分数据集，还有一个训练线性回归，主要是得到这条线的系数和截距（这里也怪怪的，主要没太懂）

第二部分是特征选择和单一模型训练，主要内容是通过上面处理好的数据来选择最能影响的五个特征，采用了两种方法分别训练，第一种是纯计算对每个特征衡量两个变量之间的线性相关程度，第二种是基于单变量线性回归的F统计量和对应的p值，调用量scikit-learn内部函数，其实不知道能不能用（），同样选出F统计量最大的五个特征。对两种方法训练森林回归器，比较那种特征选择方法可以让随机森林学的更好，输出里的柱状图就是这样得出来的。（这一部分也很难受啊啊啊啊啊啊）
这一部分还有模型融合的模块，定义了融合的权重，然后和单一的模型比较，最终分析融合效果。（更诡异的来了）

最后一部分是extended层，这一部分主要是最后画热力图，（悄悄调了一下参数，感觉调完更好看，但是不符合作业要求），以及统计方法识别极端值，移除极端值之后重训最佳模型，然后对比前后性能（嗯，其实这一段也没怎么学懂）
